
1「アプリケーションの概要」　 　大学の講義ノートや過去のテストを売買できるサービスです。

2「このサービスの機能一覧」 　
・記事投稿機能 　 　
・記事編集機能 　
・画像投稿機能（refile） 
・投稿削除機能 　
・サインアップ、ログイン機能（devise）
・ユーザ情報編集機能
・管理者機能(rails_admin)
・購入機能（pay.jp）
・ページネーション機能（kaminari） 
・いいね機能(Ajax)　
・ファイル投稿機能(carrierwave)　

3「使用した技術」 　
・HTML/CSS
・Bootstrap 
・Ruby (2.5.3)
・Ruby on Rails(Rails 5.2.4.4)
・ソース管理はGit
・heroku
・githubにアップ
